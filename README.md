The website for frontend standard.

## Getting started

Install dependencies:

``` bash
$ git clone http://10.63.32.225/eos/it-dev-spec-frontend.git
$ cd it-dev-spec-frontend
$ npm install
```

Generate:

``` bash
$ hexo generate
```

Run server:

``` bash
$ hexo server
```
