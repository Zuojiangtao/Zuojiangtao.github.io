# IT FrontEnd Standard Website
The website for frontend standard.

## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/Zuojiangtao/Zuojiangtao.github.io.git [project-name]
$ cd project-name
$ npm install
```

Run server:

```markdown
npm run serve
```

Run build:

```markdown
npm run build
```

Run clean:

```markdown
npm run clean
```

Run deploy:
```markdown
npm run deploy
```
